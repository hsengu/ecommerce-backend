## Installation
    =============
    = Git Setup =
    =============
	git clone https://github.com/hsengu/ecommerce-backend
	cd ./ecommerce-backend
	npm install

    ==================
    = MySql DB Setup =
    ==================
    mysql -u your_username -p
    source db/schema.sql
    source db/seeds.sql
